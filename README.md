# Family Members | Interactive English Website for Kids

## Project name
Family Members - Mini English Learning Platform for Kids

## Objective
Create an interactive and child-friendly website for children aged 7 to 10 to learn and practice family vocabulary in English through videos, flashcards, games, sentence practice, quizzes, and a progress system.

## Technologies
- HTML
- CSS
- Vanilla JavaScript
- No frameworks
- Ready for GitHub Pages

## Main features
- Home section with student name input
- Learn section with embedded YouTube videos
- Vocabulary cards with pronunciation using `speechSynthesis`
- Interactive flashcards
- 10 game modules:
  - Matching Game
  - Memory Game
  - Drag & Drop Family Tree
  - Bingo of Family Members
  - Unscramble the Word
  - Guess Who?
  - Balloon Pop
  - Sentence Builder
  - Family Wheel
  - Race to the Family House
- Practice section with sentence activities
- Final quiz
- Progress bar and points system
- Stars and rewards
- Progress saved with `localStorage`
- Reset progress button
- Responsive design for desktop, tablet and mobile
